# 포트폴리오

## 환경
	 * 개발 : STS 이용
	 * DB : Oracle
	 * Spring Framework, html, css, javaScript

## 적용기능
	1. CSS 템플릿 사용 - 홈페이지 전체
	2. 동영상 삽입 - 메인화면 밑
	3. 구글지도 API 이용 - Connect Us
	4. 메일 보내기 기능 - 문의게시판
	5. 이벤트 남은시간 계산 - 홈페이지 상단
	6. jQuery plugin 활용 - FAQ
	7. 이미지 슬라이드 - 메인화면 (타 홈페이지 소스 활용)
	8. 관리자 아이디 로그인은 회원관리 / (상품,공지사항,FAQ,문의답변 등) 등록 가능

## 소개
	상품구매 홈페이지이며 로그인을 통해 사이트를 이용할 수 있습니다.
	홈페이지를 보다 더 이쁘게 꾸미기 위해 css 무료템플릿을 적용했으며
	대부분의 적용기능은 공부목적을 위해 적용하였습니다.
	이미지슬라이드는 타홈페이지 소스를 거의 그대로 복사하였지만 사용에 의의를 두고 싶어 적용하였습니다.
	
	회원은 상품구매 및 주문현황 확인이 가능하며 답변을 받을 때는 등록된 이메일로 답변을 받을 수 있도록 설정하였습니다.
	관리자는 회원일때 보이지 않는 상품등록버튼, FAQ등록버튼, 답변메일보내기,회원관리 등 따로 페이지는 만들지 않고 session을 이용하였습니다.
